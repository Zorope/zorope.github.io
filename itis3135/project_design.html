Project Design


Project Overview:

  • Application and Purpose:
    The application is made to show off my clients hiking activies and is meant to find others who enjoy hiking. This application will contain photos and information about the area that they are in. Also about events my client is holding.

  • Intended Users:
    The intended users are people who will be interested in hiking in the clients area. This application will allow users to gather with others to hike with a consistent group within the area.

  • Overview of Content:
    The application will have a homepage giving a brief overview of events, info, and photos of the clubs activies. Then have an about page talking about the purpose of the website and the club. An event page talking about upcoming events. Club leader page talking about the client and their purpose. then finally a sign up form for anyone interested to join the club.
